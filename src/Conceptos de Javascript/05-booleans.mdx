# Booleans

Los booleans es una parte importante en javascript se crean utilizando las palabras clave **true** y **false**

<CodeBlock
  codeString={`
  let booleanoVerdadero = true;
  let booleanoFalso = false;
  `} 
  language='javascript'
/>

**ejemplo:**

<CodeBlock
  codeString={`
  // Declarar una variable booleana
  let miBooleano = false;

  // Definir una función que invierte el valor del booleano
    function invertirBooleano() {
      miBooleano = !miBooleano;
  }

  // Imprimir el valor inicial del booleano
  console.log(miBooleano); // Output: false

  // Llamar a la función para invertir el valor
  invertirBooleano();

  // Imprimir el nuevo valor del booleano
  console.log(miBooleano); // Output: true

  // Llamar a la función nuevamente para invertir el valor otra vez
  invertirBooleano();

  // Imprimir el valor después de la segunda inversión
  console.log(miBooleano); // Output: false
  `} 
  language='javascript'
/>

En este ejemplo, **invertirBooleano** es una función que utiliza el operador de negación lógica `!` para invertir el valor de **miBooleano**. 
Después de llamar a esta función el valor de **miBooleano** cambiará de **false** a **true** y viceversa. 
Puedes llamar a la función tantas veces como desees para invertir el booleano.

### Buenas Prácticas a la Hora de Comparar Booleans en JavaScript:

**Usa Operadores de Comparación Estricta:** JavaScript tiene operadores de igualdad débil `==` y estricta `===`. Es recomendable utilizar la igualdad estricta `===` para 
evitar conversiones automáticas de tipo y asegurarse de que tanto el valor como el tipo sean iguales.

<CodeBlock
  codeString={`
  let x = 5;
  let y = '5';

  console.log(x == y);  // true (comparación débil)
  console.log(x === y); // false (comparación estricta)
  `} 
  language='javascript'
/>

**Evita Comparaciones Redundantes:** No es necesario comparar un booleano con **true** o **false**. Simplemente usa el booleano directamente en la condición.

<CodeBlock
  codeString={`
  // Evitar esto:
  if (booleano === true) {
    // ...
  }

  // En su lugar, hacer simplemente:
  if (booleano) {
    // ...
  }
  `} 
  language='javascript'
/>
