# Map Filter y Reduce

 son funciones de orden superior en JavaScript que se utilizan comúnmente para operar sobre arreglos.

### Map:

La función "map" se utiliza para transformar cada elemento de un arreglo y crear un nuevo arreglo con los resultados de aplicar una función a cada elemento.

**ejemplo:**

<CodeBlock
  codeString={`
    // Duplicar cada elemento de un arreglo
    const numbers = [1, 2, 3, 4, 5];
    const doubledNumbers = numbers.map((num) => num * 2);
    console.log(doubledNumbers); // Output: [2, 4, 6, 8, 10]
  `} 
  language='javascript'
/>

### Filter:

La función "filter" se utiliza para crear un nuevo arreglo que contiene solo los elementos que cumplen con ciertos criterios definidos en una función.

**ejemplo:**

<CodeBlock
    codeString={`
    // Filtrar números mayores que 3
    const numbers = [1, 2, 3, 4, 5];
    const filteredNumbers = numbers.filter((num) => num > 3);
    console.log(filteredNumbers); // Output: [4, 5]
  `} 
  language='javascript'
/>

### Reduce:

La función "reduce" se utiliza para reducir un arreglo a un solo valor acumulado aplicando una función acumulativa a cada elemento del arreglo.

<CodeBlock
  codeString={`
    // Sumar todos los elementos de un arreglo
    const numbers = [1, 2, 3, 4, 5];
    const sum = numbers.reduce((accumulator, current) => accumulator + current, 0);
    console.log(sum); // Output: 15
  `} 
  language='javascript'
/>
