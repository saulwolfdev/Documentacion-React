# Template Strings

Los Template Strings se refieren al uso de backticks (`) para definir cadenas de texto que pueden contener expresiones JavaScript dentro de llaves {}. Esto se conoce como interpolación de cadenas y es una característica de ECMAScript 6 (ES6).

Ejemplo básico de cómo se usan los Template Strings:

<CodeBlock
  codeString={`
  const MiComponente = ( ) => {
  const nombre = 'John Doe';

  const div = document.createElement('div');
  const p = document.createElement('p');
  p.textContent = \`Hola, mi nombre es \${nombre}.\`;

  div.appendChild(p);

  return div;
  };

export default MiComponente;

  `} 
  language='javascript'
/>


En el ejemplo la variable `nombre` está incorporada en el texto utilizando el formato `${nombre}`, que es una característica de las plantillas de cadena de JavaScript.
Esto permite la evaluación de `nombre` como una expresión JavaScript dentro de la cadena de texto. En otras palabras, puedes incluir variables o expresiones más elaboradas dentro de una cadena de texto de JavaScript utilizando esta sintaxis especial.

También puedes realizar operaciones más complejas dentro de los Template Strings:

<CodeBlock
  codeString={`
  const MiComponente = ( ) => {
  const num1 = 10;
  const num2 = 20;

  const div = document.createElement('div');
  const p = document.createElement('p');
  p.textContent = \`La suma de \${num1} y \${num2} es \${num1 + num2}.\`;

  div.appendChild(p);

  return div;
  };

  export default MiComponente;
  `} 
  language='javascript'
/>
