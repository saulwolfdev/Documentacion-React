# Eventos

Los eventos son acciones o situaciones que ocurren en el navegador como hacer clic en un elemento, cargar una página, presionar una tecla, etc.

### Event Listener (Escuchador de Eventos):

Un event listener es una función que espera y responde a un evento específico. Se asocia con un elemento HTML y se ejecuta cuando ocurre ese evento en el elemento.

<CodeBlock
  codeString={`
  const boton = document.getElementById('miBoton');

  boton.addEventListener('click', function() {  
  console.log('¡Hiciste click en el botón!');// Código que se ejecuta cuando se hace clic en el botón
  });
  `} 
  language='javascript'
/>

### Tipos de Eventos:

Los eventos en JavaScript pueden ser del DOM (Document Object Model) eventos del teclado, eventos del mouse, eventos de formulario, etc.

<CodeBlock
  codeString={`
  const elemento = document.getElementById('miElemento');

  elemento.addEventListener('mouseover', function() {
    console.log('El puntero del mouse entró en el elemento.');// código que se quiera ejecutar cuando el puntero entra en el elemento    
  });
  `} 
  language='javascript'
/>

### Burbuja de Eventos:

Los eventos en el DOM tienden a propagarse desde el elemento objetivo hasta el elemento superior (ancestro). Esto se conoce como la fase de burbujeo de eventos.

<CodeBlock
  codeString={`
  // Detener la propagación del evento
  elemento.addEventListener('click', function(event) {
   event.stopPropagation( ); // Evita que el evento se propague al elemento superior
  });
  `} 
  language='javascript'
/>

### Delegación de Eventos:

La delegación de eventos implica asignar un único manejador de eventos a un ancestro común en lugar de asignar manejadores a cada elemento individual. 
Esto es útil para elementos creados dinámicamente.

<CodeBlock
  codeString={`
  document.addEventListener('click', function(event) {
  if (event.target.tagName === 'BUTTON') {
    // Código que se ejecuta cuando se hace clic en un botón
    console.log('Se hizo click en un botón:', event.target.textContent);// Podes agregar más código aquí para realizar acciones específicas al hacer click    
    }
  });
  `} 
  language='javascript'
/>
