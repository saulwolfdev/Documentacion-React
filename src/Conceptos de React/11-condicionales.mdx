# Condicionales

 Las condicionales se utilizan para **renderizar** diferentes partes de un componente o aplicar lógica basada en condiciones. 
 se puede usar declaraciones if, operadores ternarios y funciones para manejar condicionales en tus componentes de React.

<a id="seccion1"></a>
### Operador ternario

Podes usar el operador ternario para escribir condicionales de manera más concisa.

<CodeBlock
  codeString={`
    import React from 'react';

    const ConditionalExample = (props) => {
    return (
        <div>
        {props.condition ? (
            <p>La condición es verdadera.</p>
        ) : (
            <p>La condición es falsa.</p>
        )}
        </div>
    );
    };

    export default ConditionalExample
  `} 
  language='javascript'
/>

dynamicFiles={{

import ConditionalExample from "./ConditionalExample.js";

const App = ( ) => {

  )
}

export default App;`,

    const ConditionalExample = (props) => {
    return (

<a id="seccion2"></a>
### Operador lógico &&

Este operador evalúa la expresión a la izquierda y si es verdadera devuelve la expresión a la derecha, de lo contrario devuelve null.

<CodeBlock
  codeString={`
  const UserInfo = ({ user }) => (
    <div>
      {user && <p>Hello, {user.name}!</p>}
    </div>
  );
  `} 
  language='javascript'
/>

<DynamicSandpack
dynamicFiles={{    
"/App.js": 
`import React from "react";
import UserInfo from "./UserInfo.js";

const App = () => {
  const user = { name: 'John' };

  return (
    <div>
      <h1>Welcome to My App</h1>
      <UserInfo user={user} />
    </div>
  );
};

export default App;`, 

"/UserInfo.js":
`const UserInfo = ({ user }) => (
    <div>
      {user && <p>Hello, {user.name}!</p>}
    </div>
  );

    export default UserInfo`,
  }}
  
  dynamicVisibleFiles={["/App.js", "/UserInfo.js"]}
/>

<a id="seccion3"></a>
### Operador lógico ||

Este operador se utiliza para proporcionar un valor por defecto si la expresión a la izquierda es falsa.

<CodeBlock
  codeString={`
  const DefaultMessage = ({ message }) => (
    <p>{message || 'No message available'}</p>
  );
  `} 
  language='javascript'
/>

<DynamicSandpack
dynamicFiles={{    
"/App.js": 
`import React from "react";
import DefaultMessage from "./DefaultMessage.js";

const App = () => {
  return (
    <div>
      <DefaultMessage message="Este es un mensaje desde App" />
      <DefaultMessage message="" />
    </div>
  );
};

export default App;`, 

"/DefaultMessage.js":
`const DefaultMessage = ({ message }) => (
    <p>{message || 'No hay mensaje'}</p>
  );

    export default DefaultMessage`,
  }}
  
  dynamicVisibleFiles={["/App.js", "/DefaultMessage.js"]}
/>

<a id="seccion4"></a>
### Declaraciones if

 Se puede usar declaraciones if dentro del método render de un componente de React para decidir qué renderizar en función de ciertas condiciones.

 <CodeBlock
  codeString={`
  const NumberDisplay = ({ number }) => {
    if (number > 0) {
      return <p>Positive Number: {number}</p>;
    } else if (number < 0) {
      return <p>Negative Number: {number}</p>;
    } else {
      return <p>Zero</p>;
    }
  };
  `} 
  language='javascript'
/>

<DynamicSandpack
dynamicFiles={{    
"/App.js": 
`import React from "react";
import NumberDisplay from "./NumberDisplay.js";

const App = () => {
  return (
    <div>
      <NumberDisplay  />
      <NumberDisplay  number={1} />
      <NumberDisplay  number={-1} />
    </div>
  );
};

export default App;`, 

"/NumberDisplay.js":
` const NumberDisplay = ({ number }) => {
    if (number > 0) {
      return <p>Positive Number: {number}</p>;
    } else if (number < 0) {
      return <p>Negative Number: {number}</p>;
    } else {
      return <p>Zero</p>;
    }
  };
  
  export default NumberDisplay`,
  }}
  
  dynamicVisibleFiles={["/App.js", "/NumberDisplay.js"]}
/>

### Uso de Switch

Cuando se tiene múltiples condiciones podemos utilizar un bloque switch.

<CodeBlock
  codeString={`
  const DayOfWeek = ({ day }) => {
    switch (day) {
      case 1:
        return <p>Monday</p>;
      case 2:
        return <p>Tuesday</p>;
      // ... otros casos
      default:
        return <p>Invalid day</p>;
    }
  };
  `} 
  language='javascript'
/>

<DynamicSandpack
dynamicFiles={{    
"/App.js": 
`import React from "react";
import DayOfWeek from "./DayOfWeek.js";

const App = () => {
  return (

      <DayOfWeek  day={1}/>

    </div>
  );
};

export default App;`,

`const DayOfWeek = ({ day }) => {

        return <p>Invalid day</p>;

  dynamicVisibleFiles={["/App.js", "/DayOfWeek.js"]}
