# JSX (JavaScript XML)

JSX es una extensión de la sintaxis de JavaScript que se parece al XML o HTML pero se utiliza con React para describir cómo debería verse la interfaz de usuario. 
JSX facilita la creación de elementos de React de manera más concisa y legible.
