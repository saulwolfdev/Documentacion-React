# Props

Las "props" (propiedades) son una parte fundamental de la arquitectura de React.
Se utilizan para pasar datos desde un componente principal a sus componentes secundarios, lo que facilita la comunicaci칩n entre ellos y la personalizaci칩n del comportamiento
de los componentes secundarios.
